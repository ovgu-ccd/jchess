Tile jchess.Board.getTile ( int  a,
int  b 
)

Access a Tile of a board by axial coordinates as explained in the following article: http://www.redblobgames.com/grids/hexagons/ the Tiles are ordered differently due to inner workings of the picking system Tile( 0, 0 ) is the top most tile Tile( 1, 0 ) is left bellow Tile( 0, 1 ) is right bellow the top most tile and so forth the third cubical coordinate can be computed with b - a, while for this one negative values are valid All the Tiles on the center column have the cubical coordinate c = 0 the columns to the left are negative, and to the right are positive

Parameters:
afirst axial coordinate
bsecond axial coordinate
